Technical Overview
==================

Key Features
------------
* **Local & Secure:** 모든 데이터 처리가 로컬에서 작동하여 보안이 우수합니다.
* **Meaning-Based Search:** Sentence-BERT를 활용한 의미 기반 검색을 지원합니다.
* **Multi-Format Support:** PDF, PPTX, DOCX, TXT 등 다양한 포맷을 지원합니다.
