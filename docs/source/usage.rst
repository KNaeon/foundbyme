Usage Guide
===========

Search Scenario: Exam Preparation
---------------------------------

1. **Upload**: Upload your lecture notes (PDF/PPT) via the upload button. The system will auto-index them.
2. **Search**: Enter a natural language query in the search bar.
   
   * *Example*: "When is the assignment due?"
   
3. **Result**: FoundByMe retrieves the specific page containing the answer (e.g., "Deadline: Dec 15").
4. **Visualization**: Click the 'Galaxy View' tab to see how your query relates to other documents in the vector space.

Web Interface
-------------
* **Home**: Main search bar and introduction.
* **Docs**: List of indexed files.
* **Graph**: 3D Visualization of your knowledge base.
